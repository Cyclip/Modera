Modera bot

An adept large-scale Discord moderation assistance bot which utilises Artificial Intelligence.

Automatically detect toxic, abusive and inflammatory messages

To boost effectivity, Modera uses production-grade machine learning models to analyse the content of each message, determining whether it violates the server's policies.

Configuring the bot to tailor it to your server is designed to be as simple as possible, whilst preserving flexibility. Once you set it up, Modera will consistently monitor server activity to automate content moderation.
